Blog中插入音乐、视频、图片（Hexo+Next）

发表于 2020-03-14 更新于 2020-03-15 阅读次数： Valine： 1.5k 1 分钟
本着“有官方教程绝不可劲bb”的原则，以下问题官文很好懂的会直接甩链接，接好。

本文使用的环境是Hexo4.2和Next7.7.2

¶1 音乐

¶1.1 blog内音乐

¶1.1.1 安装hexo-tag-aplayer插件

npm install --save hexo-tag-aplayer

¶1.1.2 在站点配置文件_config.yml中敲入

¶1.1.3 blog中敲入代码{% meting id, server, type%}就可使用 MetingJS 播放器了

{% meting id, server, type%}格式解释:

  • id就是歌曲或播放列表的id，打开音乐的网页地址栏的数字就是id
  • serve就是在netease, tencent, kugou, xiami, baidu中选你的平台
  • type就是在song, playlist, album, search, artist选对应的类型。

想高级设置进阶的参见hexo-tag-aplayer中文文档中 MeingJS 支持 (3.0 新功能) 栏目

¶1.2 侧边栏音乐

在\themes\next\layout_macro\sidebar.swig中加入一行网易云外链，外链链接直接网页访问你的歌单旁边就有。

外链这种方法有个弊端，就是在歌单内播到版权音乐时会自动停止。本来侧边栏也想用上面blog的aplayer插件，但是失败告终，所以侧边栏只能用外链方法将就。

¶3 图片

借鉴Hexo官方插图教程搞定！

设置站点配置文件_config.yml中post_asset_folder: true，之后在Blog中敲入代码{% asset_img 文件名.jpg 显示的小标题 %}就ok了。

附：

不推荐![](/images/image.jpg)的方法，说实话，我的显示不了。
